# My Rest API

 A Rest API written in TypeScript, with a MongoDB database, integrated with Mongoose

## Concepts

* REST API principals
  * CRUD
  * HTTP methods
* MSC Software Architecture
* JWT & refresh tokens
* Request validation

## Technologies

* TypeScript
* Node.js
* MongoDB with Mongoose
* Express.js
* Zod validation
